# QuestBot

T# QuestBot

Telegram-бот для квестов, викторин, ролевых игр и приключений.

## Что умеет

- Прохождение квестов и заданий
- Сохранение прогресса пользователя
- Разные концовки и нелинейный сюжет
- Система очков / достижений
- Кастомные команды и меню

## Технологии

- Node.js (ES Modules)
- Telegraf (Telegram Bot API)
- Prisma + PostgreSQL
- ESLint + Prettier
- Nodemon
